# 学生管理系统
这是一个循环迭代项目，用不同的技术栈来实现的学生管理系统。
功能点，先自己想一下，然后看老师怎么实现，然后自己再实现一遍。

本次的技术栈：html + css + jQuery

## 知识点
- 获取表单数据 
$('#form').serialize() 拿到的是表单内name和value 字符串拼接
$('#form').serializeArray() name和value的数组


- 为什么存储出生年份而不是年龄？
因为年龄会变，每一年都要重新更新数据。


- 拼接对象 $.extend({},{})  Object.assign({},{})


- .show() .hide()


- 插件 组件 模块化
插件：可以独立出来用
组件：页面的一部分
